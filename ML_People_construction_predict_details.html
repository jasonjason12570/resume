Tool to change different MSSQL and workspace settings

When we work in this company, we have to make customized settings for each customer, with our own settings and development environment. So this is why I developed this application to improve and speed up the development process. 我們在這家公司工作的時候，要為每個客戶作客製化設定，有自己的設置和開發環境。所以這就是我開發這個應用程序以改進和加速開發過程的原因。

A tool to easy change the MSSQL DB and the workspace with different IP and settings. 一種使用不同 IP 和設置輕鬆更改 MSSQL DB 和工作區的工具。
